# EcoStay Backend

## How to run backend locally

1. Clone the repository.
2. Open the backend folder.
3. Run:

   ```
   npm install
   ```
4. Create a `.env` file and add:

   ```
   PORT=5000
   ```
5. Start the server:

   ```
   node server.js
   ```
6. Open:

   ```
   http://localhost:5000
   ```
