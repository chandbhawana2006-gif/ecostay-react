# Week 7 - AI Feature Prompts Log

## Prompt Variation 1

### Prompt
Create an eco-friendly travel itinerary based on the user's destination, number of days, and interests.

### Example Input
- Destination: Rishikesh
- Days: 3
- Interests: Adventure, Nature

### Example Output
A 3-day itinerary including places to visit, eco-friendly tips, local food recommendations, and a daily schedule.

---

## Prompt Variation 2

### Prompt
Act as a travel guide and recommend sustainable tourist attractions, activities, and local experiences for the given destination.

### Example Input
- Destination: Nainital
- Days: 2
- Interests: Sightseeing, Photography

### Example Output
Recommendations including Naini Lake, Eco Cave Gardens, local cafés, walking tours, and sustainable travel practices.

---

## Prompt Variation 3

### Prompt
You are an expert eco-tourism travel assistant.

Create a personalized eco-friendly travel itinerary.

Include:
- Day-wise itinerary
- Places to visit
- Local food recommendations
- Eco-friendly travel tips
- Estimated budget

### Example Input
- Destination: Rishikesh
- Days: 3
- Interests: Adventure, Nature

### Example Output
A detailed 3-day eco-friendly itinerary with activities, attractions, food recommendations, sustainability tips, and estimated travel budget.

---

# Best Prompt

Prompt Variation 3 produced the best results because it provided detailed instructions to the AI and generated a well-structured itinerary. It included daily activities, tourist attractions, local food suggestions, sustainability recommendations, and an estimated budget. The response was comprehensive, personalized, and useful for travelers, making it the most suitable prompt for the EcoStay application.
